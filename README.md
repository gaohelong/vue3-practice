### Vue cli
#### 安装
> [官方手册](https://cli.vuejs.org/zh/guide/installation.html)
```dart
npm install -g @vue/cli
OR
yarn global add @vue/cli
```

#### 创建项目
```dart
1、vue create 项目名称

2、选择需要的vue版本及配置
```
